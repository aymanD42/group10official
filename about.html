Group 10 - Web Development Project

This website is a group project for the CNS241 Group (10) course at Jamhuriya University of Science & Technology - Faculty of Computer & IT. It includes five pages that demonstrate HTML5 and CSS3 concepts.

Project Overview

The website consists of five major pages: Home, Gallery, Booking, Help, and About. Each page was handled by a specific group member and styled using Flexbox, semantic HTML, and responsive CSS techniques. The project is hosted live using GitHub Pages.
